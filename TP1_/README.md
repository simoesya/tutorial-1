# 1. Introduction

Este trabalho prático consiste no desenvolvimento de vários exercícios
utilizando a linguagem **Kotlin**.

O objetivo principal é consolidar conhecimentos sobre:

-   sintaxe básica de Kotlin
-   criação de programas simples em consola
-   organização de código em diferentes ficheiros
-   utilização de funções e estruturas básicas da linguagem

O projeto inclui vários exercícios independentes que demonstram
diferentes funcionalidades da linguagem Kotlin.

------------------------------------------------------------------------

# 2. System Overview

O sistema desenvolvido consiste num conjunto de programas Kotlin
executados em ambiente de consola.

Cada exercício encontra-se organizado em ficheiros separados dentro do
projeto.

### Estrutura geral

-   Exercício 1
-   Exercício 2
-   Exercício 3

Cada exercício resolve um problema específico utilizando a linguagem
Kotlin.

# 3. Architecture and Design

O projeto segue uma estrutura simples de projeto Kotlin.

### Estrutura do projeto

src/ └── main/ └── kotlin/ ├── Main.kt └── cm/ ├── exer_1/ │ └──
exer_1.kt ├── exer_2/ │ └── exer_2.kt └── exer_3/ └── exer_3.kt

### Componentes

**Main.kt**\
Ficheiro principal responsável por iniciar a execução do programa.

**exer_1.kt, exer_2.kt, exer_3.kt**\
